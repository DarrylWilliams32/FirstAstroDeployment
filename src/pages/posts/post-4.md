# Day 4 was a mixed bag

**Published on:** 2026-03-18

Today I learned about how to make reuseable componets in Astro. It's not much different than creating components in other frontend frameworks like Vue or Angular. Much of the way that you write them is similar. In this instance, Astro components are blocks of HTML that can be called as a custom HTML tag. This allowed me to save a ton of time when creating the header nav bar as well as the footer social media bar. If you've never created reusable components, Astro makes it super simple.
